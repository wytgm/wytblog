fontsome 动态图标的使用

添加方式很简单，在主题配置文件或者自己的配置文件中的inject处引入一个 css 库：

https://cdn.jsdelivr.net/gh/sviptzk/StaticFile_HEXO@latest/butterfly/css/font-awesome-animation.min.css

动画图标官方地址预览:https://l-lin.github.io/font-awesome-animation/
在 DOM 元素的类名添加相应的动画即可。
例如网址导航栏可以写为: 聚宝盒: /box/ ||fab fa-xbox faa-spin animated
